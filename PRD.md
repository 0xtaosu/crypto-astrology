## 1. 项目背景与概述 (Background & Overview)

### 1.1 项目背景

加密货币市场具有高度波动性、全天候交易和不可预测性。投资者在面对巨大的市场不确定性时，往往寻求技术分析（TA）和基本面分析（FA）之外的心理支撑。传统玄学（如紫微斗数、星盘）在华人及全球社区中拥有广泛的受众基础，将“玄学”与“炒币”结合，既是一种独特的娱乐方式，也能满足部分用户的心理投射需求。

### 1.2 产品定位

**Crypto Astrology** 是一款基于 Web 的轻量级娱乐化分析工具。它利用紫微斗数和西方占星学理论，结合加密货币的链上数据（合约地址、发币时间），为用户提供一份带有“赛博道教”美学风格的运势分析报告。

### 1.3 核心价值

- **情绪价值**：为焦虑的投资者提供一种另类的“确定性”或心理安慰。
- **娱乐性**：通过独特的“赛博道教”视觉风格，创造话题性和传播度。
- **决策辅助**：提供基于玄学视角的风险提示（如“大凶”、“宜观望”）。

## 3. 功能需求说明 (Functional Requirements)

### 3.1 首页与数据输入

**用户故事**：作为用户，我希望输入币种的基本信息，以便系统进行分析。

- **输入字段**：
    - **Token Name (币种名称)**：必填，字符串。
    - **Contract Address (合约地址)**：必填，字符串（用于生成唯一哈希指纹）。
    - **Genesis Date (发币时间)**：必填，精确到分钟（用于排盘）。
- **交互要求**：
    - 输入框需具备“终端命令行”风格。
    - 点击“启动占卜”按钮后，需展示 2-3 秒的“解码/计算”动画，增强仪式感。

### 3.2 玄学分析引擎 (Astrology Engine)

**用户故事**：系统需根据输入的时间和地址，自动计算运势数据。

- **紫微斗数模块**：
    - 根据发币日期计算**命宫主星**（决定项目基本面，如“紫微”代表龙头，“贪狼”代表波动）。
    - 根据发币月份计算**财帛宫主星**（决定资金流向和盈利能力）。
    - 生成**运势评分**（0-100分）和**吉凶判定**。
- **西方星盘模块**：
    - 根据发币日期计算**太阳星座**（Sun Sign）。
    - 生成**市场情绪预测**（Bullish/Bearish）。
- **哈希运势修正**：
    - 利用合约地址的 Hash 值作为随机种子，微调运势评分，确保每个币种的分析结果具有唯一性。

### 3.3 分析报告展示

**用户故事**：作为用户，我希望看到一份直观、详细且视觉风格独特的分析报告。

- **核心指标展示**：
    - **Fortune Score (运势评分)**：动态仪表盘展示。
    - **Risk Level (风险等级)**：Low / Medium / High / Extreme。
    - **Investment Advice (投资建议)**：简短有力的建议（如“重仓出击”、“建议远离”）。
- **详细解读卡片**：
    - **紫微分析卡**：展示主星、详细判词、幸运元素（颜色、数字）。
    - **星盘分析卡**：展示星座、相位影响。

### 3.4 报告导出

**用户故事**：作为用户，我希望将分析结果保存为 PDF，以便存档或分享。

- **功能要求**：
    - 点击“Download Report”按钮。
    - 生成包含币名、核心评分、关键分析结论的 PDF 文件。
    - 文件名格式：`{TokenName}_report.pdf`。

## 4. UI/UX 设计规范 (Design Guidelines)

### 4.1 设计风格：赛博道教 (Cyber-Taoism)

- **核心理念**：高科技反乌托邦与古老东方神秘主义的结合。
- **关键词**：霓虹、符咒、故障艺术 (Glitch)、虚空、仪式感。

### 4.2 视觉规范

- **色彩系统**：
    - **背景**：Deep Void Black (`#050505`)
    - **主色 (Tech)**：Neon Cyan (`#00F0FF`) —— 用于边框、数据、科技元素。
    - **强调色 (Mystic)**：Electric Purple (`#BC13FE`) —— 用于玄学概念、魔法效果。
    - **警示色**：Cinnabar Red (`#FF2A2A`) —— 用于风险提示、符咒文字。
- **字体系统**：
    - **标题**：`Orbitron` (科幻、宽体)。
    - **正文/数据**：`JetBrains Mono` (代码感、冷静)。
    - **玄学术语**：`Ma Shan Zheng` (书法、传统)。
- **组件样式**：
    - **卡片**：切角设计 (Cut corners)，半透明磨砂玻璃背景，发光边框。
    - **按钮**：带有扫描线动画，Hover 时触发“充能”光效。

## 5. 技术架构 (Technical Architecture)

- **前端框架**：React 19 + TypeScript + Vite。
- **样式库**：Tailwind CSS v4 + shadcn/ui (定制化)。
- **核心算法库**：
    - `lunar-typescript`：用于农历转换和干支计算。
    - `jspdf`：用于前端生成 PDF 报告。
- **部署环境**：静态 Web 托管 (Static Web Hosting)。

## 6. 未来规划 (Roadmap)

### Phase 2: 实时化与社交化

- **API 集成**：接入 CoinGecko/DexScreener API，自动获取代币价格和创建时间，减少用户手动输入。
- **社交分享卡片**：生成适合 Twitter/Instagram Story 的 9:16 图片，包含二维码和核心运势。

### Phase 3: 深度玄学与 AI

- **AI 解读**：接入 LLM (如 GPT-4)，根据盘面数据生成更自然、更具“神棍”口吻的详细运势文章。
- **更多流派**：增加八字（四柱）分析、易经六爻占卜功能。
- **用户系统**：允许用户保存关注的币种，每日推送运势更新。
